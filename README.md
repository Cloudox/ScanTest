# ScanTest
学习使用iOS及以后自带的扫描二维码、条形码类库
## 介绍
本工程制作了一个扫码demo，通过两个界面来达成共功能。
第一个界面提供一个按钮和一个label。
点击按钮跳转到扫码界面，在扫码界面上绘制了方框、横线动画以及说明文字，并限定了扫码范围为方框内。
当扫码成功后，会将解析出的内容返回给第一个界面由label显示出来。
